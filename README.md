# Le Burger Libération — Site web

Site vitrine pour **Le Burger Libération**, kebab & burgers à Le Mans.

## Infos client
- **Adresse :** 127 Avenue de la Libération, 72000 Le Mans
- **Téléphone :** 09 80 93 48 06
- **Domaine :** [leburgerliberation.fr](https://leburgerliberation.fr) (registrar : LWS)
- **Hébergement :** Cloudflare Pages
- **Stack :** HTML/CSS/JS statique single-file

## Déploiement
1. Push sur ce repo GitHub
2. Cloudflare Pages auto-déploie depuis la branche `main`
3. URL preview : `le-burger-liberation.pages.dev`
4. Domaine custom : `leburgerliberation.fr` (à configurer dans Cloudflare → Custom domains, et changer les nameservers chez LWS vers Cloudflare)

## Structure
- `index.html` — page unique avec `<style>` et `<script>` intégrés
